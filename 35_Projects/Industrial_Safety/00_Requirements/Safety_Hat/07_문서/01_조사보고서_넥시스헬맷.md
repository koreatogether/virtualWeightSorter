## 3. 아두이노 구현 가능성 분석

### 3.1 구현 가능한 기능 (높은 가능성)

#### 3.1.1 충격 감지 시스템
- **센서**: MPU6050 (가속도계/자이로스코프)
- **구현 난이도**: ★★☆☆☆
- **예상 비용**: 약 5,000원
- **특징**: 
  - 3축 가속도 및 각속도 측정
  - 충격 임계값 설정 가능
  - 실시간 모니터링

```arduino
// 예시 코드 구조
#include "MPU6050.h"
MPU6050 mpu;

void detectImpact() {
  Vector rawAccel = mpu.readRawAccel();
  float magnitude = sqrt(pow(rawAccel.XAxis, 2) + 
                        pow(rawAccel.YAxis, 2) + 
                        pow(rawAccel.ZAxis, 2));
  
  if (magnitude > IMPACT_THRESHOLD) {
    triggerEmergencyAlert();
  }
}
```
