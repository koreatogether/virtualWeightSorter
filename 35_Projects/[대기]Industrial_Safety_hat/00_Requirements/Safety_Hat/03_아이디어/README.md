# 아두이노 구현 아이디어

## 목적
아두이노로 실제 구현 가능한 영역중 가장 빨리 도달 할 수 있는 아이디어 부터 어려운 아이디어까지 정리.

## 아이디어 영역
- 아두이노 기반 안전 기능
- 저전력 통신 방식
- 간단한 센서 조합
- 효율적인 알림 시스템

## 폴더 구조
- `기본안전기능/` - MPU6050 충격감지, LED 경고등
- `통신아이디어/` - ESP32 WiFi, 블루투스 활용
- `센서조합/` - 온습도, 가스, 소음 센서 통합
- `알림시스템/` - 부저, 진동, LED 매트릭스
- `전력절약/` - 슬립모드, 배터리 최적화
- `간단한UI/` - 버튼, 스위치 기반 조작
