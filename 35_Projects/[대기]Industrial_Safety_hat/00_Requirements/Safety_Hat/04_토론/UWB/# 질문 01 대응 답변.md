# 질문 01 대응 답변

네, 아두이노 환경에서 UWB(Ultra Wideband) 모듈을 활용하는 예제가 있습니다.  
대표적으로 Decawave사의 DWM1000, DWM1001 모듈을 아두이노에 연결하여 실내 위치 측정, 거리 측정 등에 활용할 수 있습니다.

아두이노 UWB 활용 예제(기본 거리 측정):

```cpp
// 예제: DWM1000 UWB 모듈을 이용한 거리 측정
// 라이브러리: DW1000 by thotro (https://github.com/thotro/arduino-dw1000)
#include <SPI.h>
#include <DW1000.h>

void setup() {
  Serial.begin(115200);
  DW1000.begin();
  DW1000.select(10); // CS 핀
  DW1000.setup();
  Serial.println("DW1000 초기화 완료");
}

void loop() {
  // 거리 측정 및 결과 출력 (실제 구현은 송수신 역할에 따라 다름)
  // 자세한 예제는 라이브러리의 examples 폴더 참고
}
```

- 실내 위치 추적, 거리 측정, 태그/앵커 방식의 실시간 위치 시스템(RTLS) 구현이 가능합니다.
- 자세한 예제와 회로도는 DW1000 라이브러리의 [GitHub](https://github.com/thotro/arduino-dw1000) 및 Decawave 공식
