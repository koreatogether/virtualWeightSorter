# 아두이노 관련 기술 동향

## 목적
아두이노 호환 센서 및 모듈의 최신 기술 동향 파악

## 조사 기술
- 아두이노 호환 센서
- 통신 모듈
    1. 노이즈 캔슬 기능 필요
    2. 블루투스 기능
    https://momanx.com/blogs/moman-ideas/best-bluetooth-helmet-communication-for-safe-rides
- 전력 관리 기술
- 아두이노 IDE 및 라이브러리

## 파일 구조
- `센서모듈/` - 가속도, GPS, 온습도 등 센서 동향
- `통신모듈/` - ESP32, WiFi, 블루투스 모듈
- `전력관리/` - 배터리 관리 및 절전 기술
- `개발환경/` - 아두이노 IDE, 라이브러리 업데이트
- `호환성/` - 다양한 아두이노 보드 호환성
- `성능최적화/` - 메모리 및 처리 성능 개선
