### 1차 하드웨어 준비물
1. 아두이노 R4 WIFI 보드 1개
2. DS18B20 센서 모듈 1개 ( 각종 기능 테스트 되는것 )

### 소프트 웨어 준비물
1. 아두이노 IDE 2.XX 버전
2. DS18B20 라이브러리 선정 : 
    ```cpp
    #include <OneWire.h>
    #include <DallasTemperature.h>
    // DallasTemperature 깃허브 주소 : https://github.com/milesburton/Arduino-Temperature-Control-Library
    ```

진행 과정

1. DS18B20 센서 모듈 과 라이브러리 기본 기능에 대한 테스트를 실행 한다. ( )
    - 각종 예제중에서 프로젝트에 쓸 것들만 골라서

2. 아두이노 클라우드 과 보드 연결 가능 체크 ( )
3. DS18B20 센서 모듈에서 나온 데이타를 아두이노 클라우드로 보내기 ( )
4. 아두이노 클라우드 앱에서 대시보드 확인 해서 기본 디자인 하기 
5. 현장: 씽크대 밑에 설치 해서 와이파이 통신 강도 이상없는지 확인하고 살펴보기 ( ) 
6. 통신 이상없으면 배관에 DS18B20 센서 부분 부착을 해보기 ( )
7. 아두이노 클라우드로부터 받은 데이타를 엑셀 또는 스프레드시트에 넣고선 데이타에서 정보로 변환 하는것에 고민해보기 ( )
