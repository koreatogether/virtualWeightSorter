# 1_가장간단한예제_문자랜덤_로컬이름실시간변경 설명

이 문서는 `1_가장간단한예제_문자랜덤_로컬이름실시간변경.ino` 코드의 기능과 구조에 대해 설명합니다.

## 개요

이 코드는 ArduinoBLE 라이브러리를 사용하여 BLE 주변장치(Peripheral) 역할을 수행하는 예제입니다. 기기 이름(광고 이름)을 주기적으로 변경하며, 광고 이름으로 사용되는 단어는 `hi`, `hello`, `good`, `bad`, `poll` 순으로 변경됩니다. 변경된 단어는 시리얼 모니터에 출력됩니다.

## 주요 기능

- **BLE 초기화**: BLE 기능을 초기화하고, 실패 시 무한 루프로 멈춥니다.
- **광고 이름 설정**: 초기 광고 이름을 `words` 배열의 첫 번째 단어로 설정합니다.
- **광고 업데이트**: 매 1초마다 광고를 중지하고, 다음 단어로 광고 이름을 변경 후 다시 광고를 시작합니다.
- **시리얼 출력**: 현재 광고되고 있는 단어를 시리얼 모니터에 출력합니다.

## 코드 구조

1. **전역 변수 및 배열**: 광고에 사용할 단어 배열과 현재 인덱스, 업데이트 간격을 저장하는 변수들을 정의합니다.
2. **setup() 함수**:
   - 시리얼 통신을 초기화합니다.
   - BLE를 초기화하고, 초기 광고 이름을 설정합니다.
   - 광고를 시작합니다.
3. **loop() 함수**:
   - 광고 업데이트 주기를 체크하여 일정 시간이 지나면 광고를 갱신합니다.
   - 새로운 단어로 광고 이름을 변경 후 재광고를 시작합니다.

## 사용 방법

1. Arduino 개발 환경에서 코드를 업로드합니다.
2. 시리얼 모니터를 열어 광고되는 단어를 확인합니다.
3. BLE 주변장치 검색을 통해 변하는 기기 이름을 확인할 수 있습니다.

## 결론

이 예제는 BLE 기능을 활용한 간단한 주변장치 예제로, 주기적인 광고 이름 변경을 통해 동적 BLE 광고를 구현하는 방법을 보여줍니다.
