# Phase 1: 난수 생성기 특성 분석

## 목표
Arduino 환경에서 사용 가능한 난수 생성기들의 편중 패턴 분석

---

## Step 1.1: 기본 난수 생성기 구현

### 구현 체크리스트
- [ ] Arduino random() 기본 내장함수 (LFSR 기반) 구현
- [ ] XorShift32 32비트 경량 PRNG (RAM 4바이트만 사용) 구현  
- [ ] Hybrid XorShift32 + analogRead(A0) 노이즈 혼합 구현

### 구현 요구사항 체크리스트
- [ ] 각 생성기별 500-700g 범위 균등분포 구현
- [ ] seed 설정 방식 통일 (millis() + analogRead()) 적용
- [ ] 메모리 사용량 측정 함수 포함

---

## Step 1.2: 편중 패턴 분석 (10,000세트 테스트)

### 테스트 구조 체크리스트
- [ ] 각 생성기별로 12개씩 10,000세트 생성 로직 구현
- [ ] 목표 구간 설정: [500-540], [540-580], [580-620], [620-660], [660-700]

### 측정 항목 체크리스트
- [ ] **구간별 분포도**: 각 구간 선택 빈도 측정 (이상적: 20% ±5%)
- [ ] **연속 유사값**: ±20g 이내 연속 발생 빈도 측정
- [ ] **극단 편중**: 한 구간에 6개 이상 집중되는 케이스 수 측정
- [ ] **통계 지표**: 표준편차, 엔트로피, 균등성 지수 계산

---

## Step 1.3: 인간 패턴 시뮬레이터 추가

### 추가 생성기 구현 체크리스트
- [ ] **숙련자 모델**: 가우시안 분포 + 균등화 보정 구현
- [ ] **초보자 모델**: 특정 구간 선호(580-620g 60%) + 피로도 누적 구현
- [ ] **랜덤 작업자**: 매 100세트마다 패턴 변경 구현
