## Step 3.1: 데이터 집계 시스템

### Arduino 메모리 효율적 데이터 구조 구현 체크리스트
- [ ] **successCount**: 성공 횟수 (0-10000) uint16_t 구현
- [ ] **totalError**: 누적 오차 (평균 계산용) uint32_t 구현
- [ ] **maxError**: 최대 오차 uint16_t 구현
- [ ] **avgTimeMs**: 평균 실행시간 uint16_t 구현
- [ ] **maxTimeMs**: 최대 실행시간 uint16_t 구현
- [ ] **memoryPeakKB**: 최대 메모리 사용량 uint8_t 구현
- [ ] **timeoutCount**: 5초 초과 횟수 uint8_t 구현

### 실시간 통계 수집 체크리스트
- [ ] **Rolling Average**: 메모리 절약을 위한 누적 평균 계산 구현
- [ ] **Percentile 추정**: P95, P99 실행시간 추정 구현
- [ ] **Arduino 시리얼 출력**: CSV 형태로 PC에 전송 구현
