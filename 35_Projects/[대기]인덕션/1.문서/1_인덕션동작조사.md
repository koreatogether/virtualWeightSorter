제품 초기 동작

1. 전원 플러그를 연결
2. 절전모드 ( 디스플레이 표시 되는거 없음 ) 상태에서 전원 버튼을 1초 이상 누르면 
   화구쪽에 0이 표시되고 나머지에서도 시간쪽은 - - 표시되고 대기상태로 된다.

하이라이트 화구 사용방법

전원버튼 1초이상 누르후 하이라이트 화구를 사용할 경우
1. 하이라이트 화구선택 버튼을 누르면 하이라이트 화구가 2단계 화력으로 시작된다.
2. 화력 조절은 + , - 버튼을 통해서 한다.
3. 하이라이트 화구 선택 버튼을 1초이상 누르면 가열이 취소 되고 화구가 꺼집니다.
4. 10단계부터는 P로 표시된다.

인덕션 1 또는 2를 사용할 경우
1. 인덕션 화구 선택 버튼을 누르면 7단계 화력으로 가열 시작된다.
2. 화력 조절은 + , - 버튼을 통해서 한다.
3. 해당 화구의 화구 선택 버튼을 1초간 누르면 가열이 취소되고 화구가 꺼집니다.
4. 10단계부터는 P로 표시된다.
5. 인덕션 화구 1, 2 사용시 초고온 LED가 점등 됩니다. 

각 화구의 시간 설정
1. 기본 설정은 1시간입니다. , 꺼짐 예약 기능으로 시간을 조절할 수 있습니다.

설정한 가열 시간전에 요리가 완료되면 화구선택 버튼을 1초이상 누르거나 전원 버튼을 눌러 종료 한다.

화구에 잔열이 남아 있을 경우 대기 상태 및 절전 모드에서 고온주의 경고등 H가 깜박입니다.
