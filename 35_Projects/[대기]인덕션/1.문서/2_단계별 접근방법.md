# 인덕션 사용 방법

## 제품 개요
- 3구 (2구 IH + 1구 일반) 모델
- 터치 조작부, 음성 안내, 다양한 안전 및 보호 기능 탑재

## 기본 조작 방법
1. 전원 플러그 연결 후, 전원 버튼을 1초 이상 눌러 대기모드 진입
2. 디스플레이 및 LED 표시로 제품 상태 확인
3. 대기 모드 진입 후 일정시간 지난후 입력이 없으면 절전 모드로 진입 
4. 절전 모드는 모든 화면이 소등 , 전원 , 정지 / 잠금 버튼 제외 모든 버튼이 작동하지 않음

## 화구 사용법
### 인덕션 화구 (1, 2)
- 터치 버튼으로 화구 선택 시 기본 7단계 화력 시작
- 화력은 +, - 버튼으로 조절. 1초간 누르면 가열 취소
- 사용 시 초고온 LED가 점등됨

### 하이라이트 화구
- 전원 버튼 1초 이상 후 선택하면 2단계 화력으로 시작
- 화력 조절은 버튼으로 수행, 1초 이상 누르면 종료
- 10단계부터는 P로 표시됨

## 기능 및 설정
- 설정 버튼: 음성 볼륨, 온도/조명, 냥이 안전 기능 등 모드 설정
- 꺼짐 예약: 기본 1시간 설정을 변경 가능 
- 일시 정지: 정지/잠금 버튼으로 가열 중 정지 후 재개

## 안전 주의사항
- 화구 잔열 감지 시 고온 경고 (H 깜박임)
- 이상 작동 시 에러 코드 표시 (U, EuF, FAn, C1/C7 등)
- 제품 사용 후 반드시 전원 차단 및 냉각 상태 확인
