## 제품 구성
- 본체
- 조작부 (버튼 4개)
- 디스플레이 표시부 (7-세그먼트 2개)
- 음성 알림 LED

## 조작부 설명
1. **전원 버튼**
   - 제품의 전원을 켜고 끄는 버튼
   - 길게 누르기(1초 이상)로 동작

2. **화력 조절 버튼**
   - 화력 증가 버튼 (+)
   - 화력 감소 버튼 (-)

3. **인덕션 1구 선택 버튼**
   - 화구 선택 시 사용

4. **디스플레이 표시부**
   - 시간 표시 세그먼트
   - 화력 표시 세그먼트

## 기본 조작 방법

### 1. 전원 켜기
1. 제품의 전원 플러그를 콘센트에 연결하면 자동으로 절전 모드로 진입합니다.
2. 절전 모드에서는:
   - 모든 화면이 꺼져있습니다
   - 음성 LED가 꺼져있습니다
   - 전원 버튼만 동작합니다

### 2. 대기 모드로 전환
1. 절전 모드에서 전원 버튼을 1초 이상 길게 누릅니다.
2. 대기 모드 진입 시:
   - 시간 표시부에 "----"가 표시됩니다
   - 화력 표시부에 "---0"이 표시됩니다
   - 음성 LED가 켜집니다

### 3. 절전 모드로 전환
- 대기 모드에서 전원 버튼을 다시 1초 이상 길게 누르면 절전 모드로 전환됩니다.
- 모든 디스플레이가 꺼지고 음성 LED도 꺼집니다.

## 화력 조절 방법

### 1. 화구 선택
1. 대기 모드에서 인덕션 1구 선택 버튼을 누릅니다.
2. 선택 시 초기 화력은 7단계로 설정됩니다.

### 2. 화력 조절
- **화력 증가**: + 버튼을 누릅니다 (최대 10단계)
- **화력 감소**: - 버튼을 누릅니다 (최소 0단계)
- 화력 표시부에 현재 화력 단계가 표시됩니다

### 3. 화구 끄기
- 화력을 0으로 낮추거나
- 전원 버튼을 길게 눌러 절전 모드로 전환합니다

## 안전 주의사항
1. 사용하지 않을 때는 반드시 절전 모드로 전환해주세요.
2. 화력 조절 시 급격한 변화를 피해주세요.
3. 전원 코드가 제대로 연결되어 있는지 확인해주세요.
4. 조작부에 물이 닿지 않도록 주의해주세요.
5. 어린이가 임의로 조작하지 않도록 주의해주세요.
