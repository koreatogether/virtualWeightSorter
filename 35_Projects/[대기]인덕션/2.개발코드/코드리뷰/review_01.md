# 코드 리뷰: 4_전원_대기모드진입.ino

## 파일 설명
이 파일은 인덕션의 전원 버튼을 1초 이상 눌렀을 때 대기 모드로 진입하는 기능을 구현한 코드입니다. 
주요 구성 요소는 Arduino Uno 보드, TM1627 모듈 7-Segment 4-Digit Display, 버튼, LED 등입니다.

## 주요 기능
1. **Setup 함수**
   - 시리얼 통신을 시작하고, 7 세그먼트 모듈을 초기화하며, 버튼 디바운스 시간을 설정합니다.
   - LED 핀 모드를 설정합니다.

2. **controlLED 함수**
   - 음성 LED를 켜거나 끄는 기능을 합니다.

3. **timeSegmentDisplay 함수**
   - 시간 세그먼트에 "----"를 표시합니다.

4. **inductionSegmentDisplay 함수**
   - 인덕션 세그먼트에 "---0"을 표시합니다.

5. **powerButtonPressed 함수**
   - 전원 버튼이 눌렸을 때 모드를 1로 설정합니다.

6. **loop 함수**
   - 버튼 루프를 실행하고, 전원 버튼이 눌렸는지 확인한 후, 대기 모드로 진입합니다.

## 코드 개선 사항
1. **주석 추가**
   - 각 함수와 주요 코드 블록에 주석을 추가하여 코드의 가독성을 높였습니다.

2. **시리얼 출력 추가**
   - 코드 실행을 추적할 수 있도록 시리얼 출력 코드를 추가했습니다.

3. **함수 간 줄 간격 조정**
   - 함수 간 줄 간격을 2줄로 조정하여 코드의 가독성을 높였습니다.

## 순서도
```mermaid
td
    A[Setup 시작] --> B[시간 세그먼트 초기화 완료]
    B --> C[인덕션 세그먼트 초기화 완료]
    C --> D[버튼 디바운스 시간 설정 완료]
    D --> E[LED 핀 모드 설정 완료]
    E --> F[Loop 시작]
    F --> G{전원 버튼 눌림 확인}
    G -->|Yes| H[모드: 1로 설정]
    H --> I[대기 모드 진입]
    I --> J[시간 세그먼트: ----]
    I --> K[LED 켜짐]
    I --> L[인덕션 세그먼트: 0]
    G -->|No| F
    F --> M[버튼 루프]
    M --> G
```
