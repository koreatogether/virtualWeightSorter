# 가정용 아쿠아포닉스 시스템

## 프로젝트 개요

가정에서 운영 가능한 소형 아쿠아포닉스 시스템입니다. 금붕어 4마리와 상추 8포트를 재배하는 친환경 순환 농법 시스템을 구축합니다.

**아쿠아포닉스란?** 물고기 양식(Aquaculture)과 수경재배(Hydroponics)를 결합한 시스템으로, 물고기의 배설물이 식물의 영양분이 되고, 식물이 물을 정화하여 물고기에게 다시 공급되는 자연 순환 구조입니다.

## 시스템 사양

- **물고기**: 금붕어 4마리
- **식물**: 상추 8포트
- **운영 방식**: 실내 가정용 소형 시스템
- **관리 수준**: 초보자 친화적

## 필요한 하드웨어

### 1. 마이크로컨트롤러 보드
- **Arduino Uno** 또는 **ESP32** (1개)
  - ESP32 추천 이유: WiFi 내장으로 원격 모니터링 가능
  - 예산: 약 10,000~15,000원

### 2. 센서 (최소 구성)

| 센서 | 용도 | 수량 | 예상 가격 |
|------|------|------|-----------|
| **DS18B20** 방수 온도 센서 | 수온 측정 | 1개 | 3,000~5,000원 |
| **DHT22** 온습도 센서 | 실내 온습도 측정 | 1개 | 5,000~7,000원 |
| **수위 센서** (Float Switch) | 물탱크 수위 감지 | 1개 | 2,000~3,000원 |

**선택 사항 (업그레이드 시):**
- pH 센서: 20,000~30,000원
- TDS(전기전도도) 센서: 10,000~15,000원
- 조도 센서: 2,000~3,000원

### 3. 액추에이터 및 장비

| 장비 | 용도 | 수량 | 예상 가격 |
|------|------|------|-----------|
| **수중 펌프** (12V, 300~600L/h) | 물 순환 | 1개 | 8,000~15,000원 |
| **에어펌프** (어항용) | 산소 공급 | 1개 | 10,000~20,000원 |
| **에어스톤** | 공기 분산 | 2개 | 2,000~4,000원 |
| **5V 릴레이 모듈** (2채널) | 펌프/조명 제어 | 1개 | 3,000~5,000원 |
| **LED 식물등** (Full Spectrum) | 식물 생장 | 1~2개 | 15,000~30,000원 |

### 4. 기타 자재

#### 수조 및 재배 베드
- **물고기 수조**: 60L 이상 (60cm x 30cm x 30cm 어항)
- **재배 베드**: 플라스틱 박스 (60cm x 40cm x 15cm) - 8포트 상추용
- **그로우 베드용 자갈**: 팽창 점토 또는 화산석 5~10L
- **넷 포트** (그물망 화분): 8개

#### 배관 및 연결
- **실리콘 호스** (내경 12~16mm): 3~5m
- **PVC 파이프** (16~20mm): 필요 시 1~2m
- **호스 밴드/클램프**: 4~6개
- **T자 분배기**: 1~2개

#### 전기 및 제어
- **12V/5V 전원 어댑터**: 2~3개
- **점퍼 와이어**: 암수/수수 각 10개
- **브레드보드**: 선택 사항 (프로토타입용)
- **방수 케이스**: 아두이노 보드 보호용

#### 기타
- **타이머 콘센트**: 조명 자동 제어 (선택)
- **차광막**: 조류 번식 방지용
- **온도계/pH 측정 스틱**: 수동 측정용

## 시스템 설계

### 물 순환 구조

```
┌─────────────────┐
│   어항 (60L)    │ ← 정화된 물 유입
│  금붕어 4마리    │
└────────┬────────┘
         │ 물고기 배설물 포함 물
         ↓
    ┌────────┐
    │ 수중펌프 │
    └────┬───┘
         │
         ↓
┌─────────────────┐
│  재배 베드       │
│  (상추 8포트)    │ → 식물이 질소화합물 흡수
│  자갈/팽창점토   │
└────────┬────────┘
         │ 정화된 물
         ↓
┌─────────────────┐
│   어항으로 복귀   │
└─────────────────┘
```

### 전기 배선 (간략)

```
[12V 어댑터] → [릴레이1] → [수중펌프]
[12V 어댑터] → [에어펌프]
[12V 어댑터] → [릴레이2] → [LED 식물등]

[Arduino/ESP32]
  ├─ DS18B20 (수온센서) → GPIO 4
  ├─ DHT22 (온습도) → GPIO 5
  ├─ 수위센서 → GPIO 6
  ├─ 릴레이1 제어핀 → GPIO 7
  └─ 릴레이2 제어핀 → GPIO 8
```

## 설치 가이드

### 1단계: 하드웨어 조립

1. **어항 설치**
   - 수평이 맞는 안정적인 위치에 어항 배치
   - 에어펌프와 에어스톤 설치
   - 물 60L 채우기 (수돗물 사용 시 24시간 정치하여 염소 제거)

2. **재배 베드 설치**
   - 어항 위 또는 옆에 재배 베드 위치 설정
   - 자갈/팽창점토로 베드 채우기 (깊이 10~12cm)
   - 넷 포트 8개를 일정 간격으로 배치

3. **배관 연결**
   - 수중펌프를 어항 바닥에 설치
   - 호스로 펌프 출구와 재배 베드 입구 연결
   - 재배 베드 배수구에서 어항으로 돌아오는 호스 연결
   - 모든 연결부에 호스밴드로 고정

### 2단계: 센서 및 제어 시스템 설치

1. **센서 설치**
   - DS18B20 방수 센서를 어항에 넣기
   - DHT22 센서를 어항 근처 공기 중에 배치
   - 수위 센서를 적절한 높이에 설치

2. **아두이노/ESP32 연결**
   - 센서들을 보드에 연결
   - 릴레이 모듈을 펌프 및 조명에 연결
   - 보드를 방수 케이스에 넣기

3. **소프트웨어 업로드**
   - `src/` 폴더의 아두이노 코드 업로드
   - 시리얼 모니터로 센서 값 확인

### 3단계: 시스템 사이클링 (질소 순환 구축)

> **중요**: 물고기를 넣기 전 2~4주간 시스템을 가동하여 유익한 박테리아를 증식시켜야 합니다.

1. **초기 사이클링**
   - 물만 채운 상태로 펌프 가동
   - 암모니아 소스 추가 (생선 조각 또는 순수 암모니아)
   - 질산화 박테리아가 암모니아 → 아질산염 → 질산염으로 변환
   - 수질 테스트로 암모니아/아질산염이 0에 가까워질 때까지 대기

2. **물고기 투입**
   - 사이클링 완료 후 금붕어 4마리를 소량씩 투입
   - 첫 주는 소량만 먹이 공급 (하루 1~2회, 소량)

3. **식물 이식**
   - 상추 모종을 넷 포트에 이식
   - 뿌리가 물에 닿도록 조정

## 모니터링 및 유지보수

### 일일 점검 사항

- [ ] 물고기 건강 상태 확인
- [ ] 상추 생장 상태 확인
- [ ] 물 순환 정상 작동 확인
- [ ] 센서 값 확인 (시리얼 모니터 또는 앱)

### 주간 점검 사항

- [ ] 수위 점검 및 증발된 물 보충
- [ ] 물 탁도 확인
- [ ] pH 측정 (스틱 사용 시) - 적정 범위: 6.8~7.2
- [ ] 식물 잎 상태 확인 (영양 결핍 징후)

### 월간 유지보수

- [ ] 어항 부분 환수 (10~20%)
- [ ] 펌프 및 필터 청소
- [ ] 배관 막힘 확인
- [ ] 조명 청소 (먼지 제거)

### 센서 값 정상 범위

| 항목 | 적정 범위 | 조치 |
|------|-----------|------|
| **수온** | 18~24°C | 히터/냉각팬 사용 |
| **실내 온도** | 20~28°C | 환기 또는 난방 조절 |
| **실내 습도** | 50~70% | 가습기/제습기 |
| **pH** | 6.8~7.2 | pH 조정제 사용 |

## 급이 가이드

### 금붕어 먹이

- **빈도**: 하루 1~2회
- **양**: 2~3분 내 먹을 수 있는 소량
- **종류**: 금붕어 전용 사료
- **주의**: 과잉 급이는 수질 악화의 주요 원인

### 식물 관리

- **조명**: 하루 12~16시간 (타이머 사용 권장)
- **수확**: 상추는 파종 후 4~6주
- **교체**: 수확 후 새 모종 이식

## 문제 해결

### 물이 탁해요
- 과잉 급이 중단
- 부분 환수 (20%)
- 필터 청소

### 식물이 노랗게 변해요
- 철분 부족: 철분 보충제 소량 추가
- 질소 부족: 물고기 먹이 약간 증량 (주의)
- 조명 부족: 조명 시간 연장

### 물고기가 수면 근처만 있어요
- 산소 부족: 에어펌프 작동 확인
- 수온 확인: 너무 높거나 낮음

### 조류가 번식해요
- 조명 시간 단축 (12시간 이하)
- 어항에 차광막 설치
- 부분 환수
