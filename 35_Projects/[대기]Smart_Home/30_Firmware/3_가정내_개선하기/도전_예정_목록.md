# 가정 내 실용 아두이노 프로젝트 목록

## ⭐ 초급 - 즉시 실행 추천

### 1. 스마트 조명 제어 시스템
**구성**: 조도 센서 + PIR 센서 + 릴레이 모듈  
**목적**: 어두울 때 사람 감지 시 자동 점등으로 전기료 절감  
**난이도**: ★☆☆☆☆

### 2. 실내 공기질 모니터링
**구성**: MQ-135 + DHT22 + OLED 디스플레이  
**목적**: 실시간 공기질/온습도 모니터링 및 환기 알림  
**보완**: CO2 센서(MH-Z19) 추가 시 더욱 정확한 측정  
**난이도**: ★★☆☆☆

### 3. 비접촉 스마트 쓰레기통
**구성**: HC-SR04 초음파 센서 + 서보 모터  
**목적**: 손 접근 시 자동 개폐로 위생적 사용  
**주의**: 서보 모터 토크 충분한지 확인, 배터리 수명 고려  
**난이도**: ★☆☆☆☆

### 4. 화분 자동 관수 시스템
**구성**: 토양 수분 센서 + 워터펌프 + 릴레이  
**목적**: 토양 건조 시 자동 물주기로 식물 관리 자동화  
**보완**: 물탱크 레벨 센서 추가, 과습 방지 로직 필요  
**난이도**: ★★☆☆☆

### 8. 전력 사용량 모니터
**구성**: ACS712 전류 센서 + LCD 디스플레이  
**목적**: 실시간 전력 소비량 측정 및 절전 의식 향상  
**주의**: 고전압 작업 시 안전 주의, 절연 필수  
**난이도**: ★★☆☆☆

### 12. 우편물 알림 시스템
**구성**: ESP8266 + 적외선 센서 + Blynk  
**목적**: 우편함 개폐 감지 시 스마트폰 푸시 알림  
**보완**: 배터리 절전 모드(Deep Sleep) 구현 권장  
**난이도**: ★★☆☆☆

### 13. 애완동물 자동 급식기
**구성**: DS1307 RTC + 서보 모터 + 푸시버튼  
**목적**: 정해진 시간 자동 급식 및 외출 시 안심  
**보완**: 사료 잔량 감지 센서 추가  
**난이도**: ★★☆☆☆

### 19. 주방 멀티 타이머
**구성**: 로터리 엔코더 + 부저 + OLED  
**목적**: 여러 요리 동시 타이밍 관리  
**난이도**: ★☆☆☆☆

## ⚙️ 중급 - 자동화 확장

### 5. 비/바람 감지 창문 자동 제어
**구성**: 빗물 센서 + 풍속 센서 + DC 모터  
**목적**: 악천후 시 자동 창문 폐쇄로 실내 보호  
**주의**: 끼임 방지 센서 필수, 수동 해제 기능 추가  
**보완**: 리미트 스위치로 과구동 방지  
**난이도**: ★★★☆☆

### 9. 스마트 블라인드 제어
**구성**: 조도 센서 + 스테퍼 모터 + ESP32  
**목적**: 일출/일몰 자동 개폐 및 앱 원격 제어  
**보완**: 모터 토크 계산 필수, 블라인드 무게 고려  
**난이도**: ★★★☆☆

### 11. 재실 감지 에어컨 절전 제어
**구성**: PIR 센서 + DHT22 + IR LED  
**목적**: 사람 없을 때 자동 절전으로 전기료 절감  
**주의**: IR 신호 학습 필요, 직접 전원 차단은 위험  
**보완**: 타이머 기능 추가로 오작동 방지  
**난이도**: ★★★☆☆

### 14. 옷장 자동 제습 시스템
**구성**: DHT22 + 팬 + 펠티어 제습기  
**목적**: 고습도 시 자동 환기로 곰팡이 방지  
**보완**: 소형 히터 대신 펠티어 소자 권장  
**난이도**: ★★☆☆☆

### 16. 실내 수경재배기
**구성**: 풀스펙트럼 LED + 토양/수위 센서 + 워터펌프  
**목적**: 광주기/온습도/양액 자동 제어로 최적 재배  
**보완**: pH 센서 추가 시 전문적 재배 가능  
**난이도**: ★★★☆☆

### 17. 미세먼지 통합 디스플레이
**구성**: ESP8266 + PM2.5 센서 + 20x4 LCD  
**목적**: 실시간 날씨/미세먼지 정보 표시 및 경보  
**보완**: API 키 필요, WiFi 안정성 확보  
**난이도**: ★★★☆☆

### 20. 자동 조명 옷장

**난이도**: ★★☆☆☆

## 🔥 고급 - 신중한 접근 필요

**목적**: 가스 누출 조기 감지 및 자동 환기

**주의**: 기계식 키 백업 필수, 방수 처리 필요  
**보완**: 배터리 레벨 모니터링 및 저전력 경고  
**난이도**: ★★★★☆
