# Test Procedure 01: 기본 연결 및 센서 인식 확인

이 테스트는 하드웨어와 소프트웨어 간의 기본적인 통신 및 핀 설정이 올바른지 확인하는 과정입니다.

## 테스트 전 체크리스트
- [x] ESP32-C3 보드가 USB로 연결되어 있는가?
- [x]센서 신호선이 D2(GPIO 4)에 연결되어 있는가?
- [x] 시리얼 모니터가 115200 Baud로 열려 있는가?

## 수행 단계

### 1단계: 부팅 및 초기 인식 확인
1. 보드에 메시지가 출력될 때까지 기다립니다.
2. 아래와 같은 초기화 성공 메시지가 나오는지 확인합니다:
   ```
   Found 4 sensor(s). Initialization successful!
   System ready - DS18B20 sensors
   ```
3. 만약 `No sensors detected`가 나온다면 배선과 `config.h`의 `ONE_WIRE_BUS` 설정을 다시 확인해야 합니다.

### 2단계: 메인 메뉴 진입
1. 시리얼 입력창에 `m`을 입력하고 Enter를 누릅니다.
2. 메인 메뉴(`1~6번 선택지`)가 정상적으로 출력되는지 확인합니다.

### 3단계: 센서 상태 테이블 조회
1. 메뉴에서 `1`을 입력하여 `View Sensor Status Table`을 선택합니다.
2. 연결된 센서 개수(예: 4개)만큼 테이블 행이 출력되는지 확인합니다.
3. 각 센서의 `ROM 주소`와 `현재 온도`가 유효한 값(예: 25.0C)으로 나오는지 확인합니다.

## 통과 기준 (Pass Criteria)
- 4개의 센서가 모두 테이블에 표시되어야 함.
- 온도가 `-127.0`이나 `85.0`이 아닌 실제 실온 값으로 표시되어야 함.
