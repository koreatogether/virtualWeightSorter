# DS18B20 온도센서 주소값 관리 UI 구상

DS18B20 온도센서의 내장 메모리 주소값을 관리하고 교체를 쉽게 하는 UI를 만들기 위한 구성안입니다.

---

## 1. 센서 리스트 뷰
- 각 DS18B20 센서의 고유 주소값(ROM 코드)과 별칭(사용자가 지정 가능)을 테이블 형태로 표시
- 센서 추가/삭제/수정 버튼 제공

## 2. 센서 상세 정보 패널
- 선택한 센서의 현재 온도, 주소값, 등록일, 상태(정상/오류) 등 상세 정보 표시

## 3. 주소값 등록/변경 UI
- 새 센서 추가 시 자동으로 주소값 읽기 및 등록
- 기존 센서의 주소값을 수정하거나 교체할 수 있는 입력창 제공

## 4. 센서 교체 지원 기능
- 센서 교체 시, 기존 주소값과 새 주소값을 비교하여 자동으로 매핑/업데이트
- 교체 이력 관리(언제, 어떤 센서가 교체되었는지 기록)

## 5. 알림 및 오류 표시
- 센서 연결 오류, 중복 주소값, 정상 동작 여부를 알림으로 표시

## 6. 검색 및 필터
- 센서 주소값, 별칭, 상태 등으로 빠르게 검색/필터링 가능

## 7. 백업 및 복원
- 센서 주소값 및 설정을 파일로 백업/복원할 수 있는 기능

---

### UI 예시
- **PC용**: 테이블+상세패널(좌우 분할), 팝업창으로 추가/수정
- **모바일용**: 리스트+상세화면(탭 전환), 플로팅 버튼으로 추가/수정

이런 구성으로 사용자가 센서 주소값을 쉽게 관리하고, 교체·백업·복원 과정을 직관적으로 실행할 수 있습니다.

## 화면 구성 예시

---

### PC용 화면 구성

- **좌측**
  - 센서 리스트(테이블 형태)
    - 각 행에 센서의 주소값, 별칭, 상태(정상/오류) 표시
    - 행 선택 시 우측에 상세 정보 표시
    - 상단에 센서 추가/삭제/수정 버튼

- **우측**
  - 선택한 센서의 상세 정보 패널
    - 현재 온도, 주소값, 등록일, 상태 등
    - 주소값 변경/교체 입력창
    - 교체 이력(리스트)
    - 오류/알림 메시지 영역

- **상단/하단**
  - 검색/필터 입력창
  - 백업/복원 버튼
