# Test Procedure 02: 동작 기능 및 데이터 관리 검증

이 테스트는 센서 별 고유 ID 부여 및 임계값 설정 기능이 의도대로 작동하는지 확인합니다.

## 수행 단계

### 1단계: ID 자동 할당 (Auto-Assign)
1. 메인 메뉴에서 `2` (ID 관리)를 선택합니다.
2. `3` (Auto-Assign Unique IDs)을 선택합니다.
3. 센서 주소 순서대로 ID 1, 2, 3... 이 할당되는 과정을 지켜봅니다.
4. **비차단 검증**: 할당 중에 시리얼 모니터가 멈추지 않고 진행 상황이 한 줄씩 출력되는지 확인합니다.

### 2단계: 개별 임계값 설정
1. 메인 메뉴에서 `3` (Threshold Settings)를 선택합니다.
2. `1` (Set Individual Sensor Threshold)를 선택합니다.
3. 안내에 따라 1번 센서를 선택하고 상한(`30`), 하한(`20`)을 입력합니다.
4. 테이블이 다시 출력될 때 해당 센서의 `High/Low` 열에 저장된 값이 반영되었는지 확인합니다.

### 3단계: 임계값 초기화
1. 임계값 메뉴에서 `4` (Clear All Thresholds)를 선택합니다.
2. `Are you sure? (y/N)` 메시지가 나오면 `y`를 입력합니다.
3. 모든 센서의 임계값이 `NONE` 또는 `0`으로 초기화되는지 확인합니다.

## 통과 기준 (Pass Criteria)
- ID 자동 할당 후 테이블의 `ID` 열이 순차적으로 채워져야 함.
- 설정한 임계값이 재부팅 없이 테이블에 즉시 업데이트되어야 함.
- 작업 중 다른 시리얼 출력이 먹통이 되지 않아야 함(비차단 확인).
