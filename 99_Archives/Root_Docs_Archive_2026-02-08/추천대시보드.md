# 파이썬 기반 대시보드/시각화 도구 추천

1. **Streamlit** — 빠른 프로토타입·데모용(간단한 위젯, 낮은 학습 곡선)
2. **Dash (Plotly)** — 상호작용 대시보드, 생산 환경 적용 가능(Plotly 기반)
3. **Panel (HoloViz)** — 복잡한 시각화·대시보드 구성에 유용(블록 조합식)
4. **Plotly** — 대화형 그래프 라이브러리(단독 사용 또는 Dash와 통합)
5. **Bokeh** — 웹 기반 상호작용 시각화(대규모 데이터 패널에 강함)
6. **Voila** — Jupyter 노트북을 대시보드로 변환(노트북 기반 워크플로우에 최적)
7. **Gradio** — ML 모델 데모 및 간단한 입력/출력 인터페이스(빠른 공유)
8. **Altair** — 선언적 시각화(통계적 그래프에 적합)
9. **NiceGUI** — FastAPI 기반 간단 웹 UI/대시보드(빠른 프로토타이핑)
10. **Flask + htmx** — 경량 서버 기반 UI, 부분 렌더링으로 유연한 인터랙션

**그 외(운영/모니터링):**
- **Grafana** — 시계열 DB(InfluxDB/Prometheus 등) 연동 중심의 운영 모니터링 툴(알림·대시보드)

**추가 추천 라이브러리:** `holoviews`, `hvplot`, `matplotlib`, `seaborn` (시각화 보완)

| **Gradio**          | ML 데모           | 모델 데모에 특화, 공유 쉬움         | 대시보드 기능 제한         |     낮음 | 보통      | PoC               |
| **NiceGUI**         | 빠른 UI           | FastAPI 기반, 경량                  | 생태계 작음                |     낮음 | 보통      | 소규모            |
| **Dash (Plotly)**   | 상호작용 대시보드 | 풍부한 인터랙션, 생산환경 적용 가능 | 서버 설정·자원 관리 필요   |     중간 | 좋음      | 운영 가능         |
