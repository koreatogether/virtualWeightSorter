포트 연결(기존 포트 관리 ) 버튼에 대한 시나리오 

포트 관리 버튼 왼족으로 COM 포트 드랍아웃 박스가 있고 
COM 포트 체크를 2초마다 해서 새로운 포트가 있으면 드랍아웃 박스에 추가하고
포트가 없으면 드랍아웃 박스에 아무것도 표시하지 않음

다만 COM0 과 COM1은 윈도우즈 시스템이 사용하므로 DROPDOWN 박스에 표시하지 않음

사용자가 예를 들어 COM4 를 선택하고 포트 연결 버튼을 누르면 
아두이노보드랑 연결을 시킨다. 
그리고 연결이 되면 시리얼 연결중 이라고 표시하고
연결이 안되면 연결 안됨 이라고 표시한다.
