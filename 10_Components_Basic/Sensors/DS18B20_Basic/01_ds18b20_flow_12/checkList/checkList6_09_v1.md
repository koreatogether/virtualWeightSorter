# 시나리오 9: 임계값(Threshold) 개별/전체 설정 체크리스트

## 1. 준비 단계
- [v] Arduino UNO R4 WiFi 보드 연결 및 포트(COM4) 확인
- [v] DS18B20 센서 연결 및 정상 인식 확인
- [v] 최신 펌웨어 업로드 완료

## 2. 개별 임계값 설정 절차
- [v] 메뉴에서 'Threshold Settings' 진입
- [v] 'Set Individual Threshold' 선택
- [v] 센서 번호 입력 (1~N)
- [v] High/Low 임계값 입력 및 적용
- [v] 센서별 임계값 정상 반영 확인
   값이 기록된 후 센서 목록에서 정상적으로 유지됨 (ID 사라지지 않음, 소프트웨어 th/tl 배열에 저장됨, getTh/getTl 함수로 조회)
- [v] 'c' 또는 'cancel' 입력 시 정상 취소 동작 확인

## 3. 전체 임계값 설정 절차
- [v] 메뉴에서 'Threshold Settings' 진입
- [v] 'Set Global Threshold' 선택
- [v] High/Low 임계값 입력 및 전체 센서에 적용
- [v] 모든 센서에 임계값 정상 반영 확인
- [v] 'c' 또는 'cancel' 입력 시 정상 취소 동작 확인

## 4. 임계값 조회 및 초기화
- [v] 'View Current Thresholds'로 센서별 임계값 확인
- [v] 'Clear Thresholds' 선택 후 'y' 또는 'yes' 입력 시 전체 임계값 초기화 확인  (초기값으로 회귀를 뜻함 )
- [v] 'n' 또는 'cancel' 입력 시 초기화 취소 동작 확인

## 5. 예외/오류 처리
- [v] 임계값 범위(-55~125°C) 외 입력 시 오류 메시지 출력 확인
- [v] High < Low 입력 시 오류 메시지 출력 확인
- [v] 소수점 입력 시 정수로 변환되어 처리됨 (예: 35.5 입력 시 35로 저장)
- [v] 센서 미연결/ID 미설정 시 안내 메시지 출력 확인

## 6. 테스트 결과 기록
- [v] 각 단계별 정상 동작 여부를 테스트 결과 파일에 기록
- [v] 테스트 시작/종료 시간 기록 확인
