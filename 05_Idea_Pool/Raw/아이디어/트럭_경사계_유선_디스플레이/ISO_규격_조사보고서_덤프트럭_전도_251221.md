# 덤프트럭 전도/전복 관련 ISO 규격 조사 보고서

> **작성일:** 2025년 12월 21일  
> **목적:** 덤프트럭 전도/전복 및 트럭 기울어짐에 대한 국제 규격(ISO) 및 관련 안전기준 조사

---

## 1. 개요

본 보고서는 덤프트럭의 전도/전복 사고 예방과 관련된 국제 표준(ISO) 및 관련 규정을 조사한 결과를 정리한 것입니다. 트럭 경사계(Inclinometer) 개발 시 참고할 수 있는 기술적 기준과 안전 요구사항을 포함합니다.

---

## 2. 주요 ISO 표준

### 2.1 ISO 22138:2022 - 덤프트럭 틸트 테이블 테스트

| 항목 | 내용 |
|------|------|
| **표준명** | Heavy commercial vehicles — Vehicle stability during tipper body operation — Tilt-table test method |
| **발행년도** | 2022 |
| **적용범위** | 후방 및 측면 덤핑 대형 상용차량 (3.5톤 초과, N2, N3, O3, O4 카테고리) |

**주요 내용:**
- 덤프 작업 중 차량의 **횡방향 안정성(lateral stability)** 평가 방법 제공
- **틸트 테이블 테스트**를 통해 다양한 적재함 경사각에서의 전복 임계치 측정
- **한계 곡선(limit curve)** 생성으로 안전 작업 영역 설정
- 자율 건설 차량에도 적용 가능

> [!IMPORTANT]
> 이 표준은 **덤프 작업 중 안정성** 평가에 가장 직접적으로 관련된 ISO 표준입니다.

---

### 2.2 ISO 16333:2011 - 정상 상태 롤오버 임계치 테스트

| 항목 | 내용 |
|------|------|
| **표준명** | Heavy commercial vehicles and buses — Steady-state rollover threshold — Tilt-table test method |
| **발행년도** | 2011 (2004년 초판 개정) |
| **적용범위** | 대형 상용차량 및 버스 (M3, N2, N3, O3, O4 카테고리) |

**주요 내용:**
- 차량이 정상 상태 선회 시 **롤오버 없이 견딜 수 있는 최대 횡가속도** 측정
- 차량을 틸트 테이블에 올려 점진적으로 기울여 **임계 타이어 리프트** 시점 확인
- 단일 차량, 트랙터-세미트레일러 조합, 굴절 버스, 풀 트레일러 등에 적용

**테스트 방법:**
1. 차량 종축을 테이블 틸트 축과 평행하게 배치
2. 테이블을 점진적으로 기울임
3. 타이어가 테이블에서 들릴 때까지 진행
4. 안전 구속 장치로 실제 전복 방지

> [!NOTE]
> 이 표준은 **정적 상태**에서의 롤오버만 다루며, 동적 안정성 제어 시스템의 효과는 평가하지 않습니다.

---

### 2.3 ISO 19380 - 차량 무게중심 측정

| 항목 | 내용 |
|------|------|
| **표준명** | Heavy vehicles — Measurement of longitudinal, lateral, and estimation of vertical CG position |
| **적용범위** | 대형 상용차량 및 버스 |

**주요 내용:**
- 차량의 **종방향/횡방향 무게중심** 위치 결정
- **수직 무게중심** 위치 추정
- 틸트 안정성 평가의 기초 데이터 제공

---

### 2.4 ISO 11026:2010 - 동적 롤 안정성 테스트

| 항목 | 내용 |
|------|------|
| **표준명** | Heavy commercial vehicles and buses — Test method for roll stability — Open-loop test |
| **적용범위** | 대형 상용차량 및 버스 |

**주요 내용:**
- 곡선 주행 시 **동적 롤 안정성** 평가
- 오픈 루프 테스트 방법으로 실제 주행 조건 시뮬레이션

## 3. UN ECE 규정

### 3.1 UN ECE Regulation No. 111 - 탱크 차량 안정성

| 항목 | 내용 |
|------|------|
| **적용범위** | 위험물 운반용 탱크 차량 (N, O 카테고리) |
| **핵심 요구사항** | 23도 틸트 테스트에서 안정성 유지 |

**평가 방법:**
1. **틸트 테이블 테스트**: 양방향 23도에서 안정성 유지 필수
2. **계산 방법**: 4 m/s² 횡가속도에서 전복 여부 계산

---

### 3.2 UN ECE Regulation No. 13 (Annex 21) - 차량 안정성 기능

**적용 대상:**
- 트럭 (N2, N3 카테고리)
- 트레일러 (O3, O4 카테고리)
- 총중량 3,500kg 초과 차량

**안정성 기능:**
- **롤오버 제어 (Rollover Control)**
- **방향 제어 (Directional Control)**
- 전자식 안정성 제어(ESC) 시스템 포함

## 4. 한국 안전기준 (KOSHA)

### 4.1 관련 KOSHA GUIDE

| 문서 번호 | 명칭 |
|-----------|------|
| **C-114-2020** | 덤프트럭 및 화물자동차 안전보건지침 |
| **C-106-2016** | 덤프트럭 안전보건작업 지침 |
| **C-48-2012** | 건설기계 안전보건작업 지침 |

### 4.2 주요 안전 요구사항

#### 작업 전 점검사항
- 조종장치 및 브레이크 작동 상태
- 유압장치 누유/손상 여부
- 타이어 공기압 및 손상 여부
- 운행 구간 지반 평탄성 확인
- 적재 중량 준수 (과적 금지)

#### 작업 중 안전수칙
- 유도자 배치 (진출입로 및 하역 장소)
- 적재함 상승 중 운행 금지
- 경사지 정차 시 운전석 이탈 금지
- 현장 내 규정 속도 준수 (10km/h 이하)

#### 하역 장소 요구사항
- 평탄하고 견고한 지반 확보
- 연약 지반 및 급경사 회피
- 도랑 가장자리 안전 블록 설치

## 5. 트럭 경사계 개발 시 참고 기준

### 5.1 권장 경고 임계각도

| 조건 | 권장 경고 각도 | 근거 |
|------|---------------|------|
| **일반 경고** | 5~7° | 영국 티퍼 바디 설계 기준 (최소 7° 전복각 권장) |
| **위험 경고** | 15° | UN ECE R111 계산 기준의 65% 수준 |
| **긴급 경고** | 20° | UN ECE R111 틸트 테스트 기준(23°)의 87% |

### 5.2 측정 고려사항

```
┌─────────────────────────────────────────────────────────┐
│  경사계 측정 시 고려 요소                                │
├─────────────────────────────────────────────────────────┤
│  1. 횡방향 경사 (Lateral Tilt) - 측면 전복 위험         │
│  2. 종방향 경사 (Longitudinal Tilt) - 후방 전복 위험    │
│  3. 적재함 상승 각도 - 덤핑 작업 중 추가 불안정성       │
│  4. 지반 조건 - 연약 지반에서의 침하 고려               │
└─────────────────────────────────────────────────────────┘
```

### 5.3 안전 시스템 기능 요구사항

ISO 22138:2022 및 KOSHA 지침을 기반으로 한 권장 기능:

1. **실시간 경사 모니터링**
   - 횡방향/종방향 경사각 연속 측정
   - 적재함 상승 상태 감지

2. **단계별 경보 시스템**
   - 시각적 경고 (LED/디스플레이)
   - 청각적 경고 (버저/경보음)
   - 단계별 위험도 표시

3. **데이터 기록 기능**
   - 작업 이력 저장
   - 위험 상황 로그

## 6. 결론 및 권장사항

### 6.1 핵심 결론

1. **ISO 22138:2022**가 덤프트럭 작업 안정성에 가장 직접적으로 관련된 표준
2. **ISO 16333:2011**을 통해 롤오버 임계치 테스트 방법 확인 가능
3. **UN ECE R111**의 23도 틸트 테스트가 실무적 안전 기준으로 활용 가능
4. 한국에서는 **KOSHA GUIDE C-114-2020** 준수 필요

### 6.2 경사계 개발 권장사항

| 항목 | 권장 사양 |
|------|----------|
| 측정 범위 | ±45° (횡방향/종방향) |
| 측정 정밀도 | ±0.5° 이내 |
| 경고 단계 | 3단계 (주의/경고/위험) |
| 디스플레이 | 실시간 각도 표시 + 경고 표시등 |
| 업데이트 주기 | 최소 10Hz (0.1초) |
