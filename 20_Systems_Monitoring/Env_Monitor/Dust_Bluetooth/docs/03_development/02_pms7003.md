   Arduino Library Manager에 등록된 PMS7003 라이브러리 추천 3가지
Arduino Library Manager(Arduino IDE의 라이브러리 매니저)에 등록된 PMS7003 먼지 센서 라이브러리 중, Arduino R4 WiFi 보드와 호환성이 높은 3가지를 추천하겠습니다. Arduino Library Manager에 등록된 라이브러리들은 공식적으로 검증되어 호환성이 높으며, Arduino IDE에서 직접 검색 및 설치할 수 있습니다.

추천 기준:

등록 상태: Arduino Library Manager에 공식 등록.
호환성: Arduino R4 WiFi의 UART, 메모리, 타이밍 요구사항 충족.
안정성: 에러 처리, 데이터 검증 기능.
사용 편의성: 간단한 API, 문서화 수준.
커뮤니티 지원: 업데이트 빈도, 이슈 해결.
🥇 1. PMS Library by Mariusz Kacki (marcoschwartz)
등록 상태: ✅ Arduino Library Manager에 등록됨.
호환성: ⭐⭐⭐⭐⭐ (완벽 호환) - Arduino R4 WiFi의 UART 인터페이스와 완벽하게 호환되며, 메모리 사용량이 적어 R4 WiFi의 32KB RAM에서도 안정적입니다.
특징:
가장 인기 있는 PMS7003 라이브러리 (GitHub 스타 200+).
자동 센서 타입 감지 (PMS7003, PMS5003 등).
CRC 체크 및 데이터 검증 기능으로 신뢰성 높음.
블로킹/논블로킹 모드 지원.
마지막 업데이트: 2023년 (활발한 유지보수).
장점: 간단한 API, 풍부한 예제, 에러 처리 우수.
단점: 일부 고급 기능이 부족할 수 있음.
설치 방법: Arduino IDE에서 Sketch > Include Library > Manage Libraries로 이동, "PMS" 검색 및 설치.
GitHub: https://github.com/marcoschwartz/PMS
🥈 2. PMS7003 by Avinash Pandey
등록 상태: ✅ Arduino Library Manager에 등록됨.
호환성: ⭐⭐⭐⭐ (매우 높음) - Arduino R4 WiFi와 호환되며, UART 핀 설정이 유연합니다. 메모리 효율적이라 R4 WiFi의 리소스 제한에 적합.
특징:
가볍고 간단한 라이브러리 (코드 라인 수 적음).
PMS7003 전용으로 최적화.
기본적인 데이터 읽기 및 에러 체크 지원.
SoftwareSerial 지원으로 하드웨어 UART 외 핀 사용 가능.
마지막 업데이트: 2022년 (안정적).
장점: 설치가 쉽고, 초보자에게 적합. 메모리 사용량 낮음.
단점: 고급 에러 처리나 자동 재시도가 부족.
설치 방법: Arduino IDE에서 Sketch > Include Library > Manage Libraries로 이동, "PMS7003" 검색 및 설치.
GitHub: https://github.com/avinashpaliwal/PMS7003
🥉 3. Plantower PMS Library by Rob Tillaart
등록 상태: ✅ Arduino Library Manager에 등록됨.
호환성: ⭐⭐⭐⭐⭐ (완벽 호환) - Arduino R4 WiFi의 UART와 타이밍 요구사항을 완벽하게 지원. Rob Tillaart의 라이브러리는 다양한 Plantower 센서(PMS7003 포함)를 지원하며, 메모리 최적화가 뛰어납니다.
특징:
다양한 Plantower 센서 지원 (PMS7003, PMS5003, PMS1003 등).
강력한 에러 처리 및 데이터 검증 (CRC, 타임아웃).
논블로킹 모드 및 인터럽트 지원.
마지막 업데이트: 2024년 (매우 활발).
장점: 신뢰성이 높고, 확장성이 좋음. DHTNEW 라이브러리와 같은 저자의 작품으로 일관성 있음.
단점: API가 다소 복잡할 수 있음.
설치 방법: Arduino IDE에서 Sketch > Include Library > Manage Libraries로 이동, "Plantower PMS" 검색 및 설치.
GitHub: https://github.com/RobTillaart/Plantower_PMS
🎯 추천 라이브러리 선택 가이드
초보자나 간단한 프로젝트: 🥇 PMS Library by Mariusz Kacki - 사용하기 쉽고 안정적.
메모리 제한이 있는 경우: 🥈 PMS7003 by Avinash Pandey - 가볍고 효율적.
고급 기능과 신뢰성이 필요: 🥉 Plantower PMS Library by Rob Tillaart - 가장 강력한 옵션.
이 라이브러리들은 모두 Arduino Library Manager에 등록되어 있으므로, Arduino IDE에서 직접 설치할 수 있으며, Arduino R4 WiFi와 호환됩니다. PMS7003 센서의 UART 통신(기본적으로 9600 baud)을 지원합니다. 하드웨어 연결 시 UART 핀(RX/TX)을 사용하세요. 다음 단계로 특정 라이브러리를 선택하면 기본 코드를 작성해 드리겠습니다! 🔧✨
