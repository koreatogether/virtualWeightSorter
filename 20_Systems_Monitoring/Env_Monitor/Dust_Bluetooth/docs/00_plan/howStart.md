안녕하세요! 현재 고등학교 2학년 학생입니다! 저의 학교 동아리인 공학 동아리에서 아두이노 프로젝트로 공기청정기를 만들게 되었습니다. 제가 구상한 설계도와 기능을 적어두었습니다. 설계도에서 문제가 있는 곳과 해결방법이 있다면 피드백 남겨 주시면 정말 감사드리겠습니다!! (아직은 세밀한 코딩은 하지 않은 상태입니다!)

​

[구상한 기능]

 1. 블루투스를 이용하여 무선으로도 작동 가능 (핸드폰으로 조작)

 2. 셀렉터 스위치를 사용하여사용하여 블루투스 모드, 전원 OFF, 자동 공기청정기 모드를 선택 가능

 3. 미세먼지 농도와 온도, 습도를 측정하여 디스플레이로 표시 

 4. 휴대할 수 있도록 무선으로 작동함 

 5. C타입 충전기로 충전하여 사용 가능함 

<자동 공기청정기 기능>

 6. 미세먼지 농도가 일정 농도 이상으로 올라가면 빨간색 LED가 켜지며, 동시에 자동으로 공기청정기가 작동함 

 7. 미세먼지 농도가 일정 농도 이상으로 내려가면 파란색 LED가 켜지면서 자동으로 공기청정기가 꺼짐

(모스펫을 사용하여 제어)

​

{사용할 재료들}

아두이노 우노(Arduino Uno) 1개

미세먼지 센서: 미세먼지센서 GP2Y1012AU0F, 5V, 시리얼 통신) 1개

온습도 센서: DHT11 1개

디스플레이: LCD 16*2(I2C제어)

블루투스 모듈: HM-10(BLE 4.0 모듈) (아이폰 사용자라 IOS 지원되는 HM-10 선택함)

쿨링팬: 7025 팬모터 (12V) FNM-S-7025

3S 리튬 폴리머 배터리(11.1V)

충전 모듈: [SMG] USB C PD TO DC 터미널 단자 트리거 모듈 FEMALE[HPRO-0024]

DC-DC 컨버터 LM2596 DC-DC 스텝다운 벅 컨버터 모듈

모스펫: IRF520 MOSFET 드라이버 모듈

아두이노 신호등 LED

미세먼지 센서: GP2Y1012AU0F

셀렉터 스위치(3단)

저항

커패시터

온습도 센서: DHT11
