# Shorts Video Plan - Boiler 4x Monitor (Rev06)

## 📌 기획 배경 및 분석 (Algorithm Insights 2025)
**목표**: 기술적 완성도(Rev06)와 시각적 만족감(Dashboard)을 동시에 보여주어 Tech/Maker 니치 시청자의 체류 시간 증대.

### 1. Retention Hooks (초반 3초)
- **즉각적인 반응**: 불(라이터)이나 얼음을 센서에 댔을 때, 그래프가 즉시 반응하는 모습을 보여주어 "시스템이 살아있음"을 증명.
- **대조 효과 (Contrast)**: 복잡한 전선이 얽힌 브레드보드(Analogue)와 세련된 웹 대시보드(Digital)를 한 화면에 보여줌.

### 2. Viral Triggers (바이럴 요소)
- **"No IP Address" Flex**: `192.168.0.xxx`를 외우는 대신 `boiler.local`로 접속하는 mDNS 기능을 보여주어 "어떻게 한거야?"라는 호기심 및 댓글 유도.
- **ASMR/Satisfaction**: 물리 버튼을 '딸깍' 누를 때 LCD 화면이 즉시 전환되는 쾌감(Responsiveness).
- **Practical Utility**: 단순한 장난감이 아니라 "난방비 절감"이나 "효율 측정"이라는 실용적 목적 강조.

## 🎬 스토리보드 (60초 구성)

| :--- | :--- | :--- | :--- |
| **00~05s** | **[Hook]** 스마트폰 화면의 그래프가 급격히 치솟는 장면. 카메라가 뒤로 빠지며 센서에 라이터를 대고 있는 손을 보여줌. | (자막) **실시간 보일러 효율 측정?**<br>(효과음) 급박한 경고음 또는 상승음 | "원인(불) -> 결과(그래프)"의 직관적 연결로 시선 고정. |
| **05~15s** | **[Build]** Arduino R4 보드 클로즈업 -> 4개의 센서 연결 -> LCD 켜짐 (빠른 편집). | (자막) **아두이노 R4 + DS18B20 4개**<br>(자막) **배선은 복잡하지만...** | 하드웨어의 복잡함(멋짐)을 1초 단위 컷으로 빠르게 보여줌. |
| **15~25s** | **[The Magic]** 브라우저 주소창에 `boiler.local` 입력 후 엔터 -> 대시보드 로딩. 스마트폰에서 그래프를 꼬집어 줌인/줌아웃(Rev06 기능). | (자막) **IP 주소 몰라도 됨**<br>(자막) **boiler.local 접속!** | mDNS 기능 강조. 부드러운 Chart.js 애니메이션이 시각적 만족감 제공. |
| **25~35s** | **[Hardware/Software Sync]** 물리 버튼을 누르는 순간(좌측 화면), LCD 페이지가 넘어가고 웹 대시보드 데이터도 갱신되는 모습(우측 화면 PiP). | (자막) **물리 버튼으로 제어**<br>(자막) **LCD & Web 동기화** | 하드웨어와 소프트웨어의 완벽한 연동 강조. |
| **35~45s** | **[Benefit]** "Delta T" 위젯을 가리킴. 입수/출수 온도 차이를 보여줌. | (자막) **온도차(Delta T) 분석**<br>(자막) **난방 효율 체크까지** | 단순 온도계가 아닌 '분석 도구'임을 어필. |
| **45~55s** | **[Conclusion]** 전체 시스템 풀 샷 (보일러 배관에 붙은 센서들). 어두운 방에서 LCD 백라이트만 빛남. | (자막) **이거 하나면 난방비 걱정 끝?** | 감성적인 Tech 분위기 연출. |
| **55~60s** | **[CTA]** 깃허브 로고와 함께 리포지토리 이름 자막. | (자막) **소스코드는 고정 댓글에!** | 댓글 확인 유도 (알고리즘 점수 상승). |

## 🛠 촬영 준비 체크리스트
- [ ] **준비물**: 라이터(또는 얼음물), 스마트폰(대시보드용), 삼각대.
- [ ] **조명**: LCD와 스마트폰 화면이 잘 보이도록 주변 조명을 약간 어둡게 하고, 하이라이트 조명 하나만 보드 비추기.
- [ ] **화면 설정**:
    - 웹 대시보드를 **다크 모드** 스타일(현재 배경색이 밝음, 필요시 CSS 살짝 수정 추천)로 하거나 밝기를 낮춰 촬영.
    - LCD 글씨가 잘 보이도록 명암비(Contrast) 조절.
