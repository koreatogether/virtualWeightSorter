# Boiler Temp Monitoring System 🌡️
> **DS18B20 기반 4채널 입출수 온도 모니터링 및 실시간 분석 시스템**

---

## 1. 프로젝트 목적
- 보일러 및 난방 분배기의 입/출수 온도를 정밀 측정하여 에너지 효율 분석
- 온도차($\Delta T$)를 통한 열전달 효율 점검 및 시스템 밸런싱 근거 자료 확보

## 2. 하드웨어 구성
- **MCU**: Arduino Uno R4 WiFi
- **Sensors**: DS18B20 (4개, 1-Wire 디지털 방식)
- **Display**: 20x4 I2C LCD
- **Input**: 물리 버튼 1개 (화면 전환용)

## 3. 주요 기능 (Rev05 기준)
1. **듀얼 모니터링**: 현장(LCD)과 원격(웹 대시보드) 동시 확인
2. **mDNS 지원**: IP 주소 대신 `http://boiler.local`로 간편 접속 가능
3. **실시간 시각화**: Chart.js를 이용한 온도 변동 추이 그래프
3. **가변 로깅**: 1초~5시간 사이의 유연한 데이터 갱신 주기 설정
4. **자동 기록**: PC와 연동하여 7일간의 온도 데이터 자동 백업 (JSON)
5. **사용자 편의성**: 물리 버튼 하나로 현장 LCD 모드 전환

## 4. 웹 대시보드 미리보기
- **실시간 숫자 표시**: S1~S4 온도 및 $\Delta T$ 계산값
- **인터랙티브 차트**: 줌/팬 기능을 지원하는 선형 그래프
- **연결 상태**: PC 기록 서버와의 실시간 연동 상태 표시

## 5. 기대 효과
- 배관 내 슬러지나 유량 불균형으로 인한 열손실 조기 발견
- 난방비 절감 및 쾌적한 실내 온도 관리 최적화
