# 온도 모니터링 그래프 기능 체크리스트 (Function Checklist)

온도 분석 시스템의 두 가지 환경(실시간 웹 대시보드 및 오프라인 플레이어)에 대한 기능 구현 현황입니다.

---

## 🚀 1. 실시간 웹 대시보드 (Real-time Dashboard)
아두이노 웹 서버를 통해 현장에서 즉각적으로 모니터링할 때 필요한 기능입니다.

- [x] **실시간 데이터 갱신:** 설정된 주기에 따라 그래프 및 수치 자동 업데이트
- [x] **상태 위젯 (Current Stats):** 현재 $S1~S4$ 온도 및 그룹별 $\Delta T$ 수치 표시
- [x] **인터랙티브 줌 (Zoom/Pan):** Chart.js 줌 플러그인을 통한 터치/마우스 줌 지원
- [x] **연결 및 기록 상태 모니터링:** PC 수집기 연동 여부 및 네트워크 상태 표시


## 📼 2. 오프라인 로그 플레이어 (Log Player - Python)
수집된 대량의 JSON 로그를 정밀 분석하고 이상 지점을 탐색할 때 필요한 기능입니다.

- [x] **정밀 타임 탐색 (Jog Wheel):** 조그 다이얼을 이용한 프레임 단위 데이터 이동
- [x] **데이터 커서 (Crosshair):** 마우스 이동 시 수직선과 함께 모든 센서값 및 $\Delta T$ 툴팁 표시
- [x] **영역 채우기 (Area Fill):** $T_1, T_2$ 사이의 간격을 색상으로 채워 온도차를 시각적으로 강조
- [x] **내비게이션 툴바:** Matplotlib 표준 툴바를 통한 정밀 확대(Zoom), 이동(Pan), 저장 기능
- [x] **구간 통계 분석 (Range Stats):** 선택한 구간에 대한 MAX / MIN / AVG 자동 계산 위젯
- [x] **드래그 패닝 (Drag to Pan):** 마우스 드래그를 통한 직관적인 시간축 이동
- [x] **임계치 라인 (Buffer Limits):** 계획상한(Red), 계획하한(Blue) 라인 설정 및 표시
- [ ] **이벤트 주석 (Annotations):** 히터/펌프 등 제어 이벤트 발생 시점을 그래프 상에 마킹
