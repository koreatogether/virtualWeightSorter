sd 카드에 tft 파일을 넣어서 Nextion 에디터에서 업로드를 반드시 해야 넥션 에디터에서 직접 디버그 및 업로드가 가능합니다. 


1. 아두이노 나노 보드에 다음의 보드를 업로드 합니다. 
   ```
   // [하드웨어 바이패스 모드 코드]
// 이 코드는 내용은 비어있지만, 핀 설정을 초기화하여 
// USB 신호가 TX/RX 핀으로 직통 연결되도록 유도합니다.

void setup() {
  // 1. 하드웨어 시리얼 핀을 "입력(High Impedance)" 상태로 띄웁니다.
  // 이렇게 하면 아두이노 칩이 통신에 개입하지 않고
  // USB 칩(CH340 등)과 0,1번 핀이 직접 연결되는 효과가 납니다.
  pinMode(0, INPUT); // RX
  pinMode(1, INPUT); // TX
}

void loop() {
  // 아무것도 하지 않음 (MCU는 잠자기 모드나 다름없음)
}

    ```

    그런 다음 

    아두이노 나노 보드에서 gnd 와 넥션 gnd를 연결합니다. 
    외부 5v 전원을 받아서 작동을 하면 외부 5v 전원을 넥션 디스플레이의 5v에 연결합니다. ( 이때는 외부 전원 gnd 와 아두이노 보드 gnd 연결을 추가 해야 합니다. )

    아두이노 나노 보드의 5v를 활용할경우 넥션 디스플레이의 5v에 연결합니다.
