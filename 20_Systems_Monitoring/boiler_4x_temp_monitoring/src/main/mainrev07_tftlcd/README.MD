# mainrev07_tftlcd

## 프로젝트 목적
이 폴더는 [ROADMAP.md](../../docs/ROADMAP.md)의 **"1. 사용자 인터페이스 (UI/UX) - 디스플레이 장치 확장"** 항목에 명시된 계획에 따라, 기존 20x4 LCD를 넘어 **Nextion TFT LCD (NX4832T035)** 기반의 시각화 기능을 개발하기 위한 환경입니다.

## 주요 개발 목표
- [ ] Nextion NX4832T035 디스플레이 연동 (HMI 인터페이스 활용)
- [ ] Nextion Editor를 활용한 고해상도 그래픽 온도 모니터링 UI 디자인
- [ ] 시리얼 통신(UART)을 통한 Arduino-Nextion 간 데이터 송수신 구현
- [ ] 버튼 입력을 통한 화면 전환 및 정보 상세 보기 기능 확장
- [ ] 실시간 데이터의 로컬 디스플레이 가독성 개선

## 로드맵 참조
> **1. 사용자 인터페이스 (UI/UX)**
> - [ ] **디스플레이 장치 확장**: 현재 20x4 LCD 외에 OLED, TFT LCD, e-paper 등 다양한 시각화 장치 지원.
