# mainrev07_tftlcd

## 프로젝트 목적
이 폴더는 [ROADMAP.md](../../docs/ROADMAP.md)의 **"1. 사용자 인터페이스 (UI/UX) - 디스플레이 장치 확장"** 항목에 명시된 계획에 따라, 기존 20x4 LCD를 넘어 **Nextion TFT LCD (NX4832T035)** 기반의 시각화 기능을 개발하기 위한 환경입니다.

## 주요 개발 목표
- [x] Nextion NX4832T035 디스플레이 연동 (HMI 인터페이스 활용)
- [x] Nextion Editor를 활용한 고해상도 그래픽 온도 모니터링 UI 디자인
- [x] 시리얼 통신(UART)을 통한 Arduino-Nextion 간 데이터 송수신 구현
- [x] 실시간 데이터의 로컬 디스플레이 가독성 개선

## UI /UX 개선
- [x] ** 1분뒤 화면 꺼짐 및 복귀 기능**: 일정 시간 비활성 시 화면을 자동으로 끄고, 화면 터치 입력시 복귀하는 기능 추가.
- [ ] **화면 밝기 조절**: 설정 화면에 화면 밝기 조절 옵션 추가.
- [ ] **화면 전환 1P ~ 4P 전화버튼 맨 오른쪽에 구현하기** : 화면 전환 버튼을 화면 우측에 고정하여 쉽게 접근 가능하도록 개선.
- [ ] ** 그래프에서 원하는 선만 보여지게 하는 체크 박스 추가**: 여러 데이터 채널 중 사용자가 원하는 채널만 선택하여 표시할 수 있는 기능 추가.
