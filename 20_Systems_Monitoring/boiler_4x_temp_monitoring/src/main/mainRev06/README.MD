# Main System Revision 06

이 리비전은 **고급 분석 및 모니터링 시각화**가 적용된 확장 버전입니다.

## 주요 기능 (Rev06 Upgrade)
- **고급 시각화 (Delta Visualization)**:
    - **Overlay Graph**: 같은 그룹의 온도(S1 vs S2)를 겹쳐 그리고 차이 영역을 색상으로 표시.
    - **Delta Chart**: 온도 차($\Delta T$)만을 위한 전용 그래프 컴포넌트 추가.
- **지능형 모니터링**:
    - **동적 임계치**: 웹에서 경고/위험 단계를 직접 설정하고, 임계치 초과 시 시각적 알람 제공.
    - **통계 위젯**: 실시간 $\Delta T$의 최대/최소/평균 값을 즉시 계산하여 표시.
    - **이벤트 태깅**: 가동/정지/점검 이벤트를 그래프 상에 마커로 표시하여 이력 추적.
- **고정 IP 적용**: `192.168.0.29` IP를 사용하여 LCD와 웹 접근성을 고정.
- **Chart.js Plugin**: Zoom/Pan 및 Annotation 플러그인 통합.

## 기존 기능 (Rev05 계승)
- **가변 갱신 주기**: 1초부터 최대 5시간까지 데이터 수집 주기를 자유롭게 선택 가능.
- **물리 버튼(D3)**: 버튼을 눌러 4개의 LCD 화면(온도차1, 온도차2, 전체 온도, 시스템 정보)을 순환합니다.
- **기록 상태 표시**: PC 데이터 수집기와의 연결 여부를 웹 대시보드에서 즉시 확인.

## 파일 구조
- `main.cpp`: 전체 루틴 및 웹 서버 요청 처리.
- `Dashboard.h`: 웹 대시보드 HTML/JS 정의.
- `WiFiManager.h/cpp`: 네트워크 연결 관리.
- `SensorManager.h/cpp`: DS18B20 4개 센서의 주소 스캔 및 온도 측정.
- `DisplayManager.h/cpp`: 20x4 LCD 인터페이스 및 상태 표시.
- `wifiCredentials.h`: 환경 맞춤형 WiFi 및 비밀번호 설정.
