# TODO 02 — 페이지 전환 시 채널 상태 및 데이터 복구 (Persistence)

설명:
- Nextion 디스플레이에서 개별 채널(bt0~bt3) 또는 전체 채널(bt4) 버튼을 켠 상태에서 다른 페이지로 이동했다가 다시 메인 페이지로 돌아오면, 버튼 상태가 off로 초기화되고 웨이브폼의 그래프가 지워지는 현상이 발생합니다.

요구 사항:
- 페이지 전환 시에도 ESP32 내부의 `activeChannels` 상태를 유지해야 함.
- 메인 페이지로 다시 진입하는 시점을 감지하여, 활성화된 채널의 버튼 상태(`bt0.val=1` 등)를 다시 전송해야 함.
- 페이지 복귀 시 웨이브폼에 버퍼의 과거 데이터를 다시 그려주어 그래프 연속성을 확보해야 함.

검증(테스트 케이스):
- 메인 페이지에서 일부 채널(예: S1, S2) 활성화 확인.
- 다른 페이지로 이동 후 다시 복귀.
- 버튼 상태 유지 및 그래프 재출력 확인.
