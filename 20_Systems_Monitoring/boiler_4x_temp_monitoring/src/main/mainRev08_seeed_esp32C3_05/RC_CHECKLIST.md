# Rev03 릴리스 후보 체크리스트

이 체크리스트는 Rev03 릴리스 후보(RC)를 만들기 전에 완료해야 할 작업을 추적하기 위한 문서입니다.

## 우선순위: 높음 (RC 이전 필수)
- [ ] A1: WiFi 자동 재연결 구현 및 WebServer가 연결 실패 시 재시작되도록 구현
- [ ] A2: Nextion 핸드셰이크 상태기계(타임아웃/재시도) 및 화면상의 에러 표시 추가
- [ ] A3: 전력 소비 측정 리포트 완성 및 권장 PSU 스펙 문서화(권장: ≥1A)

## 우선순위: 중간
- [ ] A4: 대시보드 비밀번호 외부 설정화(config) 및 세션 타임아웃 추가
- [ ] A5: CI 빌드 작업 생성(Windows 및 Ubuntu)
- [ ] 자동화된 HIL 테스트 추가(Nextion 장시간 동작, WiFi 끊김 시뮬레이션)

## 우선순위: 낮음
- [ ] 원격 텔레메트리: 현장 장치용 전압/전류 텔레메트리 옵션 추가
- [ ] 로깅 포맷 개선 및 샘플 로그를 `progressLog.md`에 첨부

### 메모
- 작업 항목은 `raptor_checklist.json`의 ID를 참조합니다.
- 높은 우선순위 항목이 완료되면 `Rev03-rc1` 태그를 만들고, 정식 필드 릴리스 전 전체 HIL(48~72시간) 테스트를 실행하세요.
