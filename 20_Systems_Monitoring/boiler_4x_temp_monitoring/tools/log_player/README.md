# Boiler Log Player (로그 분석기)

JSON 형식으로 수집된 보일러 온도 로그 데이터를 시각적으로 재생하고, 정밀하게 분석하는 데스크톱 GUI 도구입니다.

## 📂 버전 관리 (Revisions)

이 도구는 기능 추가에 따라 버전별로 관리됩니다.

### ✅ Main Rev 02 (현재 최신 버전)
> **중점 사항**: 데이터 분석 능력 강화 (구간 통계, 상호작용성 확장)

*   **[New] 그래프 드래그 패닝 (Interactive Pan)**
    *   그래프 영역을 **마우스 왼쪽 버튼으로 드래그**하여 시간축을 직관적으로 좌우 이동할 수 있습니다.
*   **[New] 임계치 라인 설정 (Threshold Lines)**
    *   사용자가 계획상한(Red)과 계획하한(Blue) 온도를 직접 입력하여 그래프에 기준선을 표시할 수 있습니다.
    *   실제 온도와 계획 온도를 한눈에 비교 분석하기 용이합니다.
*   **[New] 구간 통계 자동 분석 (Range Statistics)**
    *   그래프를 확대(Zoom)하거나 이동(Pan)할 때, 현재 **화면에 보이는 구간**에 대한 통계가 자동으로 계산됩니다.
    *   **제공 정보**: 구간 내 온도차($\Delta T$)의 **최대값(Max)** 및 **평균값(Avg)**.
    *   **활용**: 특정 이상 발열 구간을 확대했을 때, 그 구간이 얼마나 심각했는지 수치로 즉시 확인 가능.

### ✅ Main Rev 01 (초기 완성 버전)
> **중점 사항**: 사용자 편의성 및 코드 구조화

*   **모듈형 아키텍처**: `app.py`, `components.py`, `config.py`로 코드 분리.
*   **인터랙티브 데이터 커서**: 마우스를 움직이면 수직선(Crosshair)과 함께 모든 센서의 온도값을 툴팁으로 표시.
*   **표준 내비게이션 툴바**: Matplotlib의 기본 툴바를 내장하여 **줌(Zoom Box)**, **이동(Pan)**, **저장(Save)** 기능 제공.
*   **영역 시각화**: 두 센서($S_1-S_2$, $S_3-S_4$) 사이의 간격을 색상으로 채워 온도차를 직관적으로 표현.

## 🎮 주요 기능 가이드

### 1. 조그 다이얼 (Dual Jog Wheel)
*   영상 편집기처럼 프레임 단위로 데이터를 탐색할 수 있습니다.
*   좌측 휠은 그룹 1($S_1, S_2$), 우측 휠은 그룹 2($S_3, S_4$)를 독립적으로 제어합니다.

### 2. 가변 타임라인 (View Interval)
*   10초부터 10시간까지, 한 화면에 보여줄 시간 범위를 자유롭게 조절할 수 있습니다.
*   넓게 보며 전체 추세를 파악하거나, 1분 단위로 좁혀 미세한 변화를 감지할 수 있습니다.

### 3. 구간 통계 (Range Stats) - *Rev 02 Only*
*   별도의 버튼 클릭 없이, 그래프를 확대하면 즉시 좌측 패널의 **"구간 통계"** 위젯이 갱신됩니다.
*   이상 징후가 의심되는 구간을 드래그하여 확대해보세요.

## 🚀 실행 방법

### 필수 요구 사항
*   Python 3.x
*   라이브러리: `matplotlib`, `tkinter` (보통 Python에 내장됨)

### 실행 명령어

**최신 버전 (Rev 02) 실행:**
```powershell
python tools/log_player/mainrev02/log_player.py
```

**이전 버전 (Rev 01) 실행:**
```powershell
python tools/log_player/mainrev01/log_player.py
```
