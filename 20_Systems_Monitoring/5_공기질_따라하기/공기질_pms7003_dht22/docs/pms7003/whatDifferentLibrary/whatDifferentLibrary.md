# PMS 계열 센서(PMS7003, PMS5003 등) 아두이노 라이브러리 목록

PMS 미세먼지 센서를 위한 주요 아두이노 라이브러리들과 저장소 주소입니다.

## 1. PMS Library (by fu-hsi)
*   **저장소:** [https://github.com/fu-hsi/PMS](https://github.com/fu-hsi/PMS)
*   **특징:** 가장 널리 사용되는 라이브러리입니다. PMS1003, PMS3003, PMS5003, PMS7003 등 Plantower의 PMS x003 시리즈 전체를 지원합니다. 하드웨어 시리얼과 소프트웨어 시리얼 모두 호환됩니다.

## 2. PMserial (by avaldebe)
*   **저장소:** [https://github.com/avaldebe/PMserial](https://github.com/avaldebe/PMserial)
*   **특징:** 비차단(Non-blocking) 시리얼 통신을 위해 설계된 강력한 라이브러리입니다. PMSx003 및 PMSa003 등 다양한 센서를 지원하며, ESP32/ESP8266 및 일반 아두이노 보드에서 잘 작동합니다.

## 3. pms5003 (by jbanaszczyk)
*   **저장소:** [https://github.com/jbanaszczyk/pms5003](https://github.com/jbanaszczyk/pms5003)
*   **특징:** PMS5003 센서 전용 라이브러리입니다. 구현이 간단하며 해당 모델에 특화되어 있어 PMS5003 프로젝트에 적합합니다.

## 4. Plantower_PMS7003 (by jmstriegel)
*   **저장소:** [https://github.com/jmstriegel/Plantower_PMS7003](https://github.com/jmstriegel/Plantower_PMS7003)
*   **특징:** PMS7003 센서를 타겟으로 한 라이브러리입니다. PMS7003 전용의 가벼운 구현이 필요할 때 유용합니다.
