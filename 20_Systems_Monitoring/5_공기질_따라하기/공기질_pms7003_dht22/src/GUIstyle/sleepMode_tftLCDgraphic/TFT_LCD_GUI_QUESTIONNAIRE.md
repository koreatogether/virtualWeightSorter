# PMS7003 TFT LCD GUI 디자인 설문지

OLED(`u8g2lib`)와 Processing으로 구현된 GUI를 **TFT LCD**로 이식하기 위해, 하드웨어 구성과 디자인 선호도를 파악해야 합니다.

참고한 코드:
- `sleep_Mode_Graphic_processingExample/pms7003_gui.pde` - Processing 800x600 컬러 GUI
- `sleepMode_oledGraphic/u8g2libStyle.ino` - 0.96" OLED 128x64 흑백 GUI
- `test_pms7003/sleepMode/basicExample/sleepMode_UI_example_refactored.ino` - 시리얼 기반 UI

아래 항목에 체크(x) 또는 직접 입력해 주세요.

## 2. 화면 디자인 및 레이아웃

### Q6. 화면 방향 (Orientation)
*   [x] **세로 (Portrait)**: 기본 방향
*   [ ] **가로 (Landscape)**: 옆으로 눕힌 방향

### Q7. 기본 색상 테마
Processing GUI는 다크 모드(어두운 배경)를 사용했습니다. TFT에서는:
*   [x] **다크 모드**: 검정 배경 + 밝은 텍스트 (Processing GUI와 동일)
*   [ ] **라이트 모드**: 흰색/밝은 배경 + 어두운 텍스트
*   [ ] **커스텀**: 직접 지정 (배경색/텍스트색 입력): ________________

### Q8. 화면 구성 방식
TFT는 OLED보다 크지만, 여전히 정보를 어떻게 배치할지 결정해야 합니다:
*   [x] **탭 방식 (Processing GUI와 유사)**: 상단에 탭(모니터링/설정)을 두고 터치나 버튼으로 전환
*   [ ] **수동 전환 (OLED와 유사)**: 인코더/버튼으로 뷰(Monitor/Graph/Stats/Settings) 전환
*   [ ] **단일 대시보드**: 모든 정보를 한 화면에 배치 (큰 해상도 활용)
*   [ ] **자동 순환**: 여러 화면을 일정 시간마다 자동 전환

### Q9. 막대 그래프 표현 방식
Processing GUI에서는 컬러 막대(초록/노랑/빨강)를 사용했습니다:
*   [x] **컬러 막대 (Processing과 동일)**: PM 수치에 따라 초록→노랑→빨강으로 색상 변화
*   [ ] **그라데이션 막대**: 높이에 따라 색상이 변하는 그라데이션
*   [ ] **단색 막대 + 배경색 변화**: 막대는 고정색, 위험 시 배경색 변경
*   [ ] **아이콘 + 텍스트**: 이모지/아이콘으로 공기질 상태 표시 (😊/😐/😷)

### Q10. 폰트 크기 우선순위
*   [x] **PM 2.5 강조**: PM 2.5 수치를 매우 크게, 나머지는 작게
*   [ ] **균등 표시**: 모든 PM 수치(1.0, 2.5, 10)를 비슷한 크기로
*   [x] **상태 메시지 강조**: "좋음/보통/나쁨/위험" 텍스트를 크게
