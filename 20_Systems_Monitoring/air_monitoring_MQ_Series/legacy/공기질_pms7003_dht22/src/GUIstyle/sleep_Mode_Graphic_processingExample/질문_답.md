### 화면별 구성

#### 1. 메인 화면 (Monitor)
```
┌─────────────────────────────────────┐
│ Air Monitor  ⚙️         12:34 │ 25px
├───────────┬───────────┬─────────────┤
│  PM1.0    │  PM2.5    │   PM10      │
│    12     │    35     │    42       │ 110px
│  (좋음)   │  (보통)   │  (좋음)     │
├───────────┴───────────┴─────────────┤
│   🌡️ 25.3°C     │     💧 55%       │ 80px
├─────────────────────────────────────┤
│ [메인] [설정] [그래프]              │ 25px
└─────────────────────────────────────┘
```

#### 2. 설정 화면 (Settings)
- Sleep 시간 설정: 1분, 3분, 5분, 10분, 20분
- Active ↔ Passive 모드 전환 버튼
- 데이터 리셋 버튼

#### 3. 그래프/히스토리 화면
- 히스토리 라인 그래프 (PM2.5 중심)
- 최고/최저/평균 통계 표시

### 센서 모드
- **Active Mode**: 자동 데이터 전송 + Sleep/Wake 제어 가능
- **Passive Mode**: 요청 시에만 데이터 전송
