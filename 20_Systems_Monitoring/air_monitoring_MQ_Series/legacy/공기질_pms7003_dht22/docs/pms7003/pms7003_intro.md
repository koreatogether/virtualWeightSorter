# PMS7003 소개 (Introduction)

## 주요 특징 (Main Characteristics)

*   **오경보율 제로 (Zero false alarm rate)**: 신뢰성 높은 데이터 제공
*   **실시간 응답 (Real-time response)**: 즉각적인 공기질 변화 감지
*   **정확한 데이터 (Correct data)**: 정밀한 측정값
*   **최소 식별 입자 크기**: 0.3 마이크로미터 (0.3μm)
*   **강력한 간섭 방지 성능**: 6면 차폐(Shielding) 특허 구조 적용으로 외부 간섭 최소화
*   **유연한 설계 적용**: 다양한 디자인에 맞춰 공기 흡입구(Inlet)와 배출구(Outlet) 방향 선택 가능
*   **초소형 크기 (Smallest)**: 컴팩트한 디자인

## 개요 (Overview)

**PMSA003**은 공기 중에 부유하는 입자의 개수(즉, 입자 농도)를 측정하여 디지털 인터페이스 형태로 출력하는 **디지털 범용 입자 농도 센서**입니다.

이 센서는 공기 중 부유 입자 농도와 관련된 다양한 계측기나 환경 개선 장비(공기청정기 등)에 탑재되어, 정확한 농도 데이터를 실시간으로 제공할 수 있습니다.
