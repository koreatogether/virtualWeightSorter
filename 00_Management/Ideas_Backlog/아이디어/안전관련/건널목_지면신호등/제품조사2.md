# 제품조사2: 건널목용 지면(바닥형) LED 신호등 — 제품 스펙 & 국내 사례 정리 📋

**요약**
- 해외: 철도 건널목(grade crossing)에 적용된 In‑pavement / In‑roadway Warning Lights(IRWL) 제품·사례 존재 (FRA 권고·LightGuard TraxAlert 등).  
- 국내: 횡단보도용 바닥형 보행신호등이 보편화됨(단품 가격·조달 등록 존재). 설치(토목·제어 포함) 단위당 수천만 원 사례(기사 인용).

---

## 1) 검증된 주요 제품 (스펙 요약 + 이미지/제품 페이지 링크) ✅

### 1. LightGuard Systems — TraxAlert (Rail Grade IRWL)
- 제품/용도: TraxAlert™ Advanced Grade Crossing IRWL Warning Light System — 철도 건널목 전방에 설치되는 레드/앰버 in‑roadway 경고등 시스템 (정지선·버퍼 정의, 차단기 연동).  
- 주요특징: 기존 경보/차단기와 연동, 레드(정지) / 앰버(사전경고) 조합, 시스템 컨트롤러 제공.  
- 제품 페이지(검증): https://www.lightguardsystems.com/traxalert-advanced-grade-crossing-irwl-warning-light-system  
- 이미지(예시): https://lirp.cdn-website.com/58351645/dms3rep/multi/opt/7th-Street-RR-Crossing-Elk-City-OK-1-1920w.jpg
- 비고: Elk City(OK) 등 실설치·연구 사례 보유. 문의/견적: 페이지 내 Contact


### 2. LaneLight Traffic Technologies — In‑road Stop Bar / Crosswalk systems
- 제품/용도: In‑road flashing LED systems — stop‑bar, wrong‑way, grade crossing stop‑bar 등 (도심·고속 환경 모두 적용 사례).  
- 주요특징: 적색/앰버 옵션, 원격 제어·소프트웨어(모니터링), LA Metro 등 설치 사례, 브로셔 제공(PDF).  
- 제품 페이지(검증): https://lanelight.com/products/wrong-way-and-stop-bar-systems/  
- 브로셔(예시): https://lanelight.com/wp-content/uploads/2025/06/BR0053-StopBarEnhancement.pdf  
- 이미지(예시): https://lanelight.com/wp-content/uploads/2021/06/LA-Metro-homepage-1030x456.jpg


### 3. (국내) ㈜에스에이텍 — 바닥형 보행신호등 (Floor Type Walking Signal Lamp)
- 제품/용도: 횡단보도 대기선에 설치되는 바닥형 보행신호등(보행자용 보조장치).  
- 주요특징(제품 페이지 정보 요약): 다양한 규격(크기·전원), DC24V 동작 모델, 주야간 조광·내수·미끄럼저항·보행신호 연동 등.  
- 제품 페이지(검증): http://www.sa-tech.kr/product/led  
- 비고: 교통신호 제어기 연동 옵션 보드로 실시간 통신 가능


### 4. 성풍솔레드 (SP-400) — 조달(혁신장터) 등록 제품
- 제품/용도: LED 바닥형 보행신호 보조장치 (모델명 SP-400, 제조: 성풍솔레드)  
- 가격(조달 표기): 희망가 230,000원(제품 단가, 혁신장터 표기; 납품·설치 포함 비용은 별도)  
- 조달 페이지(검증): https://ppi.g2b.go.kr:8914/sm/searchGoodsDetailView.do?invGdsIdntNo=00000047  
- 이미지(상품 상세 페이지 내 첨부 이미지 제공)
