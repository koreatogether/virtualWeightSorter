# 입간판 / 배너 쓰러짐 감지 및 알림 시스템


##  목적
- 쓰러짐을 감지하여 가게 이미지 추락 방지 

## 동기
- 간판 관리 안되는 가게는 평소 관리가 허술할 것 생각을 주게됨
## 주요 기능
  - 쓰러짐 감지 후  부저 , led , 휴대폰 알림 , 매장내 알림 

## 하드웨어 (일반하드웨어,배선도)
- 오프라인 방식
  - 아두이노 보드 + MPU-6050 + 부저 + LED 
- 온라인 방식
  - wifi / 블루투스 지원 보드 + 오프라인 알림방식 + 휴대폰 + 메일 발송등
  
## 소프트웨어
- 아두이노 라이브러리 + 센서 기본 코드 활용
- 아두이노 wifi 라이브러리 + 인벤터 / iot 클라우드 서비스 이용

## 가상시뮬레이션
